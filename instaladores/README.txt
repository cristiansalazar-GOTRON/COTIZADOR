# Cotizador de Equipos de Importacion - Instalador

## Instrucciones de Instalación

### Opción 1: Instalador Automático (Recomendado)
1. Haz doble clic en: **instalar.bat**
2. Acepta la solicitud de permisos de administrador
3. El programa se instalará automáticamente

### Opción 2: Instalación Manual
1. Copia la carpeta `dist\cotizador.exe` a: `C:\Program Files\Cotizador Automatico\`
2. Crea un acceso directo manualmente en el Menú Inicio y Escritorio

## Características del Programa

### Icono Personalizado
- **Diseño**: Caracol azul y gris con espiral
- **Tamaño**: Ampliado al doble para mejor visibilidad
- **Formato**: Icono de alta calidad con múltiples resoluciones (16x16 hasta 256x256)

### Archivo de Tasas Guardadas
- **Ubicación**: `c:\PROGRAMAS CRISTIAN\COTIZADOR AUTOMATICO\tasas_guardadas.json`
- **Funcionalidad**: Las tasas se guardan automáticamente en esta carpeta

## Ubicación de los Archivos

### Programa Instalado
- **Ubicación**: `C:\Program Files\Cotizador Automatico\`

### Archivo de Tasas Guardadas
- **Ubicación**: `c:\PROGRAMAS CRISTIAN\COTIZADOR AUTOMATICO\tasas_guardadas.json`
- Las tasas se guardarán automáticamente en esta carpeta

### Accesos Directos
- **Menú Inicio**: Cotizador Automatico > Cotizador de Equipos
- **Escritorio**: Cotizador de Equipos (si optaste por crear acceso directo)

## Desinstalación

### Opción 1: Desde Windows
1. Abre Configuración > Aplicaciones > Programas y características
2. Busca "Cotizador de Equipos de Importacion"
3. Haz clic en Desinstalar

### Opción 2: Acceso Directo
- En el menú Inicio, busca "Desinstalar" en la carpeta Cotizador Automatico

## Requisitos del Sistema
- Windows 7 o superior
- No requiere instalación de Python
- Espacio en disco: ~150 MB

## Solución de Problemas

### El instalador dice que requiere permisos de administrador
- Haz clic derecho en `instalar.bat`
- Selecciona "Ejecutar como administrador"

### El programa no se abre
- Verifica que tengas instalado .NET Framework
- Intenta desinstalar y reinstalar

### Las tasas no se guardan
- Verifica que la carpeta `c:\PROGRAMAS CRISTIAN\COTIZADOR AUTOMATICO\` exista
- Verifica permisos de escritura en esa carpeta

### El icono no se actualiza
- Presiona F5 en el Explorador de Archivos
- O reinicia el Explorador de Windows
